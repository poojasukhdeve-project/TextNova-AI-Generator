# TextNova-AI-Generator 🚀

An AI-powered text generation web application built using a transformer-based language model (nanoGPT), Flask, and NLP techniques.

---

## 📌 Project Overview

TextNova-AI-Generator is a lightweight GPT-style text generation system that generates human-like text from user prompts.

This project demonstrates:
- Transformer architecture
- Language modeling
- NLP text generation
- Temperature-based sampling
- Top-k and Top-p sampling
- Flask web integration

---

## ✨ Features

- AI text generation using nanoGPT
- Interactive Flask web application
- Temperature comparison (0.30, 0.60, 0.65)
- Top-k sampling implementation
- Top-p (nucleus) sampling
- Prompt-based text generation
- Modern dark-themed UI
- Adjustable creativity control

---

## 🛠️ Technologies Used

- Python

- NLP

---

## 📊 Model Information

- Model Type: Character-level Transformer
- Layers: 6
- Parameters: 2.6M
- Dataset: Twitter Text Dataset

---

## 🧠 How It Works

1. User enters a text prompt
2. Model predicts next tokens
3. Temperature controls creativity
4. Top-k and Top-p sampling improve output quality
5. Generated text is displayed on the web interface

---

## 🚀 Run the Project

### Install dependencies

```bash
pip install torch flask tiktoken
```

### Run Flask app

```bash
python app.py
```

Open browser:

```text
http://127.0.0.1:5000
```
